# QA Portfolio - Manual Testing Project
## Overview
This repository contains manual QA testing work completed on the SauceDemo application. The goal of this project is to demonstrate core QA skills including test case design, execution, defect reporting, and documentation.

---

## Project 001: SauceDemo Login Testing
### Application Under Test
https://www.saucedemo.com

---

## Tools Used
- Google Sheets (Test Case Management)
- Jira (Bug Tracking)
- Github (Documentation & Version Control)
- Chrome Browser (Manual Testing)

  ## Test Coverage
  The following scenarios were tested:
  - Valid login
  - Invalid login credentials
  - Empty username field
  - Empty password field
  - Both fields empty
